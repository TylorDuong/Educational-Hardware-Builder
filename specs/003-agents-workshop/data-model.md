# Data Model: Guided Weather-Station Workshop

## Reference Build Step

Ordered identity, instruction, safety category, cited lesson, optional checkpoint, and zero or more symbolic mating selections.

## Reference Template Request

An identity, associated step, template name, and bounded values for the L-bracket.

## Relationships

One plan has 12 steps. Exactly 3 steps have checkpoints. One template request belongs to one step. Mating selections refer to shared part and feature IDs; physical transforms are absent.
